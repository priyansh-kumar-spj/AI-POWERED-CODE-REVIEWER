# AI Powered Code Reviewer

A web-based AI-powered code review tool that analyzes source code and provides structured feedback, improvements, and best-practice suggestions.

This project focuses on **developer productivity**, **clean UI**, and **real-world deployment readiness**.

- Production-ready deployment setup
